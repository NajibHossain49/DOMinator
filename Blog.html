What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a tree-like structure that represents a webpage, allowing JavaScript to access and manipulate its elements and content dynamically.

How do you select an element from the DOM ?

We can select an element from the DOM using methods like "document.getElementById('id')", "document.querySelector('.class')", or "document.getElementsByClassName('class')".

what is event delegation in the context of the DOM, and why it is useful ?

Event delegation is when we add one event listener to a parent element to handle events from its child elements, making it easier and faster to manage those events.

How do you manipulate an element's attributes and styles using the DOM ?

We can manipulate an element's attributes using "element.setAttribute('attr', 'value')" and change its style with "element.style.property = 'value' " in the DOM.
